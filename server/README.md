## Description

Бэкенд написан на [Nest](https://github.com/nestjs/nest) framework.

## Installation

После перехода в папку backend

```bash
$ cd backend
```
Прописываем
```bash
$ npm install
```

## Запуск серверной части

```bash

$ npm run start:dev

```
